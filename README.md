# **iktek labs**

Bem-vindo ao repositório oficial da **iktek labs**. Este espaço é dedicado ao gerenciamento da nossa infraestrutura, fluxos de trabalho e automações no GitHub.

## **Sobre a iktek labs**

A iktek labs é uma consultoria especializada em tecnologia, focada na convergência entre hardware e software. Atuamos na intersecção de sistemas embarcados, automação industrial e desenvolvimento de soluções de software de alto nível.

## **Nossas Frentes Tecnológicas**

* **Sistemas Embarcados & Hardware Design:** Desenvolvimento e arquitetura de sistemas utilizando FPGAs (Xilinx Zynq-7000, Artix-7), SoCs e microcontroladores (STM32).  
* **Automação Industrial:** Consultoria em sistemas de controle, integração de sensores e protocolos industriais (Modbus).  
* **Desenvolvimento de Software:** Soluções escaláveis utilizando Python (LangChain, FastAPI), Node.js (React, Vite, Prisma) e C/C\#/C++.  
* **Linux & Infraestrutura:** Expertise em customização e administração de sistemas baseados em Linux (Debian) e automação de sistemas.

## **Configuração do Repositório**

Este repositório contém configurações globais para a organização iktek-labs, incluindo:

* Templates de Issues e Pull Requests.  
* Workflows de CI/CD automatizados.  
* Políticas de segurança e padrões de codificação.
